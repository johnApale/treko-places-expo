# TREKO Places Expo App
## Description:
A mobile app used internally within the TREKO startup organization. This app is meant to simplify the process of adding new place and location data to the TREKO database.
###### **NOTE:** *This app is no longer in production as it is being forked and repurposed for a different tool within the organization.*
